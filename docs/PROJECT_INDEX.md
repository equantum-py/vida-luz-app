## 🗂️ Estructura del Directorio

A continuación, se describe la organización de carpetas y archivos en el repositorio para mantener un estándar de desarrollo limpio y estructurado:

```text
vida-luz-app/
├── assets/          # Recursos estáticos (imágenes, iconos, fuentes)
├── docs/            # Documentación del proyecto (este directorio)
├── src/             # Código fuente de la aplicación

```

*(Esta estructura se irá expandiendo a medida que avance el desarrollo).*
